cardObject Documentation

Properties

data

Returns an object containing all available data about the cards.

Example


mtg.getCardByName("Elvish Mystic").data.manaCost // Returns {G}

Methods

match

Syntax : 
Boolean cardObject.match(String query)
Returns true if the card match the query named query.

Example


mtg.getCardByName("Elvish Mystic").match("c!g o:add") // Returns true
