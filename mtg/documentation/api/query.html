New syntax

Regular expressions

Use a regular expression to match with the oracle.

Examples

reg:"\{[1-9]\}"
reg:/add \{[wgrbu]\} to your mana pool/g

Abilities

Match cards with the specified ability.

Example

ab:flying

Loyalty

Match cards with the specified loyalty

Examples

loy=4
loy<4
loy>cmc

Hybrid

Match cards with hybrid mana in their mana cost (ex : {G/W})

Example

is:hybrid

Monohybrid

Match cards with monohybrid mana in their mana cost (ex : {2/W})

Example

is:monohybrid
